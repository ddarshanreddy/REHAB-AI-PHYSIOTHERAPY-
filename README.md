# REHAB-AI-PHYSIOTHERAPY-
PhysioNova is an AI-powered physiotherapy platform that uses real-time pose estimation to monitor rehabilitation exercises and provide instant posture correction and performance feedback. The system tracks metrics such as accuracy, repetitions, range of motion, and stability, while enabling physiotherapists to remotely monitor the patients exercise
All exercise data is securely stored and visualized through a dedicated analytics dashboard, enabling doctors and physiotherapists to track patient adherence, monitor recovery trends, identify incorrect movement patterns, and make data-driven treatment decisions without requiring frequent in-person visits.

Key Features
AI-powered real-time pose detection and posture correction
Guided physiotherapy exercises with visual instructions
Automatic repetition counting
Performance metrics including accuracy, ROM, stability, hold time, and session score
Personalized exercise recommendations
Patient progress tracking over time
Doctor dashboard with recovery analytics and performance reports
Secure cloud-based storage of rehabilitation sessions
Tech Stack
Frontend: React / Next.js
Backend: Node.js / Express
AI & Computer Vision: MediaPipe, TensorFlow
Database: Firebase / Supabase
Visualization: Chart.js
Authentication: Firebase Authentication
Vision

Our goal is to make physiotherapy more accessible, affordable, and data-driven by combining artificial intelligence with remote healthcare. PhysioNova helps improve exercise adherence, reduces incorrect exercise execution, enables continuous patient monitoring, and empowers healthcare professionals with actionable recovery insights.

By transforming traditional rehabilitation into an intelligent digital experience, PhysioNova aims to improve patient outcomes while supporting clinics and hospitals with scalable remote physiotherapy solutions.
